# UIUPC Project Architecture & Stack

## 🚀 Overview
The **UIUPC (United International University Photography Club)** official website is a high-performance, feature-rich web application. It serves as the primary portal for club members and administrators, featuring a sophisticated **"Studio"** suite for image processing and a robust **Admin** dashboard.

---

## 🛠 Tech Stack

### **Core Framework & Runtime**
- **Next.js 16 (React 19)**: Utilizing the App Router and Server Components.
- **TypeScript**: Full-stack type safety.

### **Styling & UI**
- **Tailwind CSS 4**: Modern CSS-first utility framework.
- **Motion (Framer Motion)**: Advanced animations and page transitions.
- **View Transitions API**: Cinematic radial shutter effects for theme switching.
- **Material 3 Principles**: Design tokens for a premium feel.

### **Backend & Infrastructure**
- **Supabase**: 
  - **Auth**: SSR-compatible authentication.
  - **Database**: PostgreSQL with RLS.
  - **Storage**: Image and document hosting.
- **SWR**: Client-side data fetching and caching.

### **State Management**
- **Zustand**: Global state for UI and the complex Studio editor.

### **Specialized Image Processing (Studio)**
- **@dnd-kit**: Drag-and-drop collage tools.
- **browser-image-compression**: Client-side optimization.
- **ExifReader & piexifjs**: Metadata manipulation.
- **react-easy-crop**: Precision cropping.
- **jsPDF & JSZip**: Portfolio generation and batch exports.

---

## 🧩 Key Architectural Patterns

1.  **Feature-Modular Architecture**: Logic is encapsulated within `src/features`, ensuring high maintainability.
2.  **Advanced Image Pipeline**: Sophisticated client-side processing reduces server overhead and improves responsiveness.
3.  **Cinematic UX**: High attention to detail with custom background grids and fluid transitions.
4.  **Security-First**: Strict Supabase RLS policies for data protection.
